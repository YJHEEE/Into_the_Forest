③휴양림/수목원
1) 휴양림/수목원/교육프로그램 조회
 - 입장료, 영업시간, 주차정보 등 정보 제공
 - 페이지 상단에 추천 테마코스 제공
 - 카카오지도api
2) 휴양림/수목원 예약
 - 휴양림 예약시 체크박스 활용해서 교육프로그램 예약하기
 - 페이지 상단에 추천 테마코스 제공
 - 네이버페이api
 - 카카오지도api

④커뮤니티
1) 명산/자전거길/등산로 방문후기작성
 - 베스트 후기 상단에 띄우기
 - 후기에 좋아요, 댓글 기능
 - 게시글 좋아요 순으로 정렬가능
 - 방문후기 작성시 방문장소 카테고리 선택가능(ex. 지리산, 설악산, 000수목원 ...)
 - 후기글에서 방문장소페이지(산림휴양/둘레길/자전거길 페이지)로 이동가능
2)명산top100 스탬프 순위
 - 최근 스탬프 많이 찍힌 순으로 정렬(1개월이내)
 - 스탬프 많이 찍힌 순으로 정렬(전체기간)
 - 스탬프 많이 찍은 회원 랭킹(3개월/6개월/전체기간)
 - 해당 페이지에서 명산 클릭시 상세페이지로 이동가능
 - 계절별 스탬프 찍힌 개수 표시 (ex. 000산 봄 11개 / 여름 102개 / 가을 46개 / 겨울 5개 ...)
3)산, 둘레길, 기타 정보 등록 및 수정 요청 게시판
 - 일반 회원만 요청게시글 작성 가능
 - 관리자가 게시글 확인 후 정보 등록 및 수정 확인표시 띄우기
 - 등록 및 수정 불가시 불가사유와 표시 띄우기

⑤마이페이지
1) 개인정보
 - 이름, 아이디, 비밀번호, 전화번호 조회
 - 비밀번호 수정
2) 예약
 - 휴양림 예약 정보 조회/삭제
3) 명산 top100 스탬프
 - 내가 찍은 명산 top100 스탬프 조회
3) 리뷰
 - 작성한 리뷰(후기) 조회/수정/삭제
 - 수정은 리뷰 작성한 페이지로 넘어가서 수정

⑥로그인
1) 회원가입
 - 일반회원만 카카오 / 네이버 api / 자체 회원가입 사용
 - 관리자(admin)은 자체 회원가입만 사용가능
2)로그인
 - 카카오 / 네이버 api / 자체 로그인 가능
